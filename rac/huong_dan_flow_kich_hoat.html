SimpleMailMessage message = new SimpleMailMessage(); message.setFrom(from); message.setTo(to); message.setSubject(subject); message.setText(text); emailSender.send(message);

Khởi tạo bì thư SimpleMailMessage: Đối tượng chuyên đựng Text đơn giản của Java Spring Mail.

Set parameters: Nhồi nhét các thông tin mà Service ở trên gửi vào (Tiêu đề, Địa chỉ đích, Nội dung url + mã token).

Gửi đi (emailSender.send): Máy chủ của bạn mở cổng mạng (port 587 bằng TLS), lấy thông tin đăng nhập từ application.properties đã phân tích, và kết nối với Máy chủ Google để bắt nó ship cái thư này đi!

public ResponseEntity<?> kichHoatTaiKhoan(String email, String maKichHoat){ NguoiDung nguoiDung = nguoiDungRepository.findByEmail(email); // ... Kiểm tra null hay Đã kích hoạt rồi bỏ qua if(maKichHoat.equals((nguoiDung.getMakichHoat()))){ nguoiDung.setDaKichHoat(true); nguoiDungRepository.save(nguoiDung); return ResponseEntity.ok("Kích hoạt tài khoản thành công"); } else { return ResponseEntity.badRequest().body("Mã kích hoạt sai!"); } }

Tìm Người Dùng Trước: Dùng cái Email nhận được từ Client, truy vấn móc từ Database con người dùng thật sự mang Email đó lên.

So Khớp Mã Bảo Mật: Đây là linh hồn của hệ thống (Dòng chữ maKichHoat.equals(...)). Ta lấy mã kích hoạt VỪA NHẬN TỪ CLIENT, so sánh có bằng với mã kích hoạt ĐÃ LƯU TRONG DATABASE TỪ BƯỚC 2 hay không.

Bật Cờ Kích Hoạt (True): Rất tuyệt vời! Nếu 2 cái mã khớp nhau, ta set setDaKichHoat(true), và cập nhật (save) lại xuống DB. Từ giờ trở đi, User này đã được cấp quyền login hợp lệ mà không bị khóa nữa.
